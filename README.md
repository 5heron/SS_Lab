## Getting Started

Clone this repo
```
$ git clone https://github.com/KTU-CSE/System-Software-lab.git
```
Switch to the program folder
```
$ cd the/dir/where/program/file/is
```
Compile the source code
```
$ gcc yourProgramFile.c
```
To Execute
``` 
$ ./a.out
```
